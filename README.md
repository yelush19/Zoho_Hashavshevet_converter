# Zoho → Hashavshevet Converter

ממיר חשבוניות מ-Zoho Books לפקודת ייבוא לתוכנת חשבשבת.

## תכונות
- המרת תאריך גרגוריאני ללוח עברי
- זיהוי אוטומטי של עמודות מקובץ Zoho
- תצוגה מקדימה של הנתונים
- ייצוא ל-CSV ו-TXT (מופרד טאבים)
- אפשרות לשנות חשבונות חובה/זכות

## הרצה מקומית

```bash
pip install -r requirements.txt
streamlit run app.py
```

## פריסה ב-Streamlit Cloud
1. העלו את הריפו ל-GitHub
2. היכנסו ל-[share.streamlit.io](https://share.streamlit.io)
3. בחרו את הריפו ואת `app.py`
